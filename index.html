Guía de Instalación de Windows en Notebooks

Este tutorial te guiará paso a paso para instalar Windows en tu notebook.

1. Preparación

Especificaciones Mínimas de Hardware:

  • Procesador: 1 GHz o más rápido
  • RAM: 1 GB (32 bits) o 2 GB (64 bits)
  • Espacio en disco: 16 GB (32 bits) o 20 GB (64 bits)
  • Tarjeta gráfica: DirectX 9 o superior con controlador WDDM 1.0
  • Pantalla: Resolución de 800x600 o superior

Materiales Necesarios:

  • Teclado y mouse configurados (si es necesario)
  • Imagen ISO de Windows
  • Unidad USB con al menos 8GB de capacidad
  • Programa para crear USB booteable (por ejemplo, Rufus)

2. Creación de un USB Booteable

Descargar e Instalar Rufus

  1. Descarga e instala Rufus desde el enlace proporcionado.
  2. Abre Rufus.

Configuración de Rufus

  1. Selecciona el Dispositivo: Asegúrate de que la unidad USB esté conectada y seleccionada en Rufus.
  2. Selecciona la ISO: Haz clic en "Seleccionar" y elige la imagen ISO de Windows.
  3. Esquema de Partición y Sistema de Archivos:
    • Esquema de partición: MBR o GPT según tu configuración BIOS/UEFI
    • Sistema de archivos: NTFS
  4. Haz clic en "Empezar" para crear el USB booteable.

3. Configuración del BIOS/UEFI

Acceso al BIOS/UEFI

  • Comúnmente, las teclas para acceder al BIOS/UEFI son F2, Del, Esc, o F10. Consulta el manual de tu notebook si no estás seguro.

Cambiar el Orden de Arranque

  1. Navega al menú de "Boot" o "Arranque".
  2. Configura la USB como la primera opción en el orden de arranque.
  3. Guarda los cambios y sal del BIOS/UEFI.

4. Instalación de Windows

Proceso de Arranque desde el USB

  1. Conecta la USB booteable a la notebook.
  2. Enciende la notebook y arranca desde el USB (debe arrancar automáticamente si configuraste el orden de arranque correctamente).

Pantalla de Carga e Instalación

  1. Selecciona el idioma, formato de hora y teclado.
  2. Introduce la clave de producto (si es necesario).
  3. Elige el tipo de instalación (personalizada o actualización).

Selección y Partición del Disco Duro

  1. Elige la unidad donde deseas instalar Windows.
  2. Si es necesario, particiona el disco.

Proceso de Copiado y Configuración Inicial

  • Windows copiará los archivos y se instalará en tu notebook. Este proceso puede tardar unos minutos.
  • Configura tu cuenta de usuario, opciones de privacidad, etc.

5. Post-Instalación y Configuración Adicional

Buscar e Instalar Actualizaciones

  1. Ve a Configuración > Actualización y seguridad > Windows Update.
  2. Busca e instala todas las actualizaciones disponibles.

Instalación de Software Adicional

  • Instala navegadores, suites de oficina, antivirus y cualquier otro software necesario.

Configuraciones Recomendadas

  • Ajusta las configuraciones de rendimiento y elimina cualquier bloatware innecesario.
